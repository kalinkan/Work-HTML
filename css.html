CSS3

CSS (англ. Cascading Style Sheets — каскадные таблицы стилей) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

Преимущественно используется как средство описания, оформления внешнего вида Web-страниц, написанных с помощью языков разметк HTML и XHTML, но может также применяться к любым XML-документам, например, к SVG или XUL
